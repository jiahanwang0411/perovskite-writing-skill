# Domain Vocabulary for Perovskite Solar Cell Research

Use these terms to maintain the elevated academic register expected in Nature/Science family journals.

## Device Physics

| Term | Abbreviation | Usage context |
|------|-------------|---------------|
| power conversion efficiency | PCE | Primary efficiency metric |
| open-circuit voltage | VOC | Voltage at zero current |
| short-circuit current density | JSC | Current per unit area at zero voltage |
| fill factor | FF | Ratio of max power to VOC x JSC |
| maximum power point tracking | MPPT | Operational stability measurement protocol |
| external quantum efficiency | EQE | Wavelength-dependent photon-to-electron conversion |
| stabilized power output | SPO | Steady-state efficiency under continuous operation |

## Stress / Strain / Thermal

| Term | Context |
|------|---------|
| biaxial tensile strain | In-plane strain from CTE mismatch |
| residual stress | Stress remaining after fabrication/annealing |
| interfacial toughness | Resistance to delamination at layer interfaces |
| fracture energy | Energy required to propagate a crack |
| adhesion energy | Energy per unit area to separate layers |
| coefficient of thermal expansion | CTE — thermal mismatch driver |
| thermal fatigue | Cumulative damage from temperature cycling |
| thermomechanical stress | Combined thermal and mechanical stress |
| thermal shock | Rapid temperature change inducing stress |
| thermal expansion compensation | Strategy to balance CTE mismatch |

## Degradation Mechanisms

| Term | Context |
|------|---------|
| grain boundary (GB) fragmentation | Mechanical breakdown at grain interfaces |
| lattice bond rupture | Breaking of crystal lattice bonds |
| defect accumulation | Progressive increase in trap states |
| delamination | Layer separation at interfaces |
| crack propagation | Extension of fractures through films |
| non-radiative recombination | Carrier loss without photon emission |
| trap-assisted recombination | Carrier loss via defect states |
| phase segregation | Compositional separation in mixed-halide perovskites |
| ion migration | Mobile ion movement under field/thermal stress |

## Mechanism-Related Terms

| Term | Context |
|------|---------|
| photoisomerization | Light-triggered molecular structural change |
| dynamic damage release | Reversible stress-relief mechanism |
| strain-release coupling | Correlation between strain and defect relaxation |
| in situ polymerization | Polymerization during device fabrication/annealing |
| defect passivation | Neutralization of electronic trap states |
| charge density difference | Electron redistribution at interfaces |
| steric repulsion | Spatial hindrance between molecular groups |
| pi-conjugation | Delocalized electron system in organic molecules |

## Stability Metrics

| Term | Definition |
|------|-----------|
| T80 | Time to 80% of initial performance retention |
| T90 | Time to 90% retention |
| n80 | Number of cycles to 80% retention |
| retention percentage | "retained X% of initial PCE" |
| degradation rate | Percentage loss per unit time/cycle |

## Elevated Academic Register Vocabulary

Use these elevated terms instead of simpler alternatives in top-tier manuscripts:

| Elevated term | Avoid | Nuance |
|--------------|-------|--------|
| resilience | stability | Connotes recovery ability, not just resistance |
| durability | lifetime | Connotes robustness under stress |
| mitigate | reduce / fix | Precise technical verb for addressing problems |
| facile | simple / easy | Emphasizes simplicity/elegance of approach |
| stringent | strict / harsh | Emphasizes rigorous test conditions |
| pristine | original / untreated | For unmodified/control samples |
| champion | best | For best-performing device |
| detrimental | harmful / bad | For harmful effects |
| adverse | negative | For negative impacts |
| substantially | significantly / greatly | Preferred in Nature journals for large effects |
| negligible | small / tiny | For near-zero changes |
| corroborate | confirm / verify | For cross-validating with multiple methods |
| rationalize | explain | For explaining observed trends theoretically |
| elucidate | show / clarify | For detailed mechanistic explanation |
| underscore | highlight | For emphasizing importance |
| impede | block / slow | For hindering processes |
| ameliorate | improve | Formal term for improvement |
| warrant | deserve | For justifying further investigation |
| underexplored | not studied | For knowledge gaps |
| pivotal | important | For critical role/significance |
